# Food-Delivery-Analytics-Dashboard
Developed a Power BI dashboard built on a multi-table data model to analyze food delivery performance, order status, revenue trends, delivery time, meal periods, payment modes, and partner-wise order distribution using DAX measures and calculated columns.

Highlights

- Tracks total orders, customers, revenue, menu count, and average delivery time.
- Shows order status distribution (Delivered, Cancelled, Pending, Returned).
- Analyzes orders by meal period (Morning, Afternoon, Evening, Night).
- Breaks down orders by delivery partners and payment methods.
- Displays revenue trends month-wise and order patterns by day.
- Categorizes delivery time using DAX-based time buckets.
- Built using 5 related tables with calculated columns and measures for accurate insights.
